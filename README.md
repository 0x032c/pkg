# go-pkg

通用 Go 项目基础工具包集合。目录如下：

- `pkg/log`        日志工具
- `pkg/config`     配置读取
- `pkg/errors`     错误封装
- `pkg/util`       常用工具函数
- `pkg/http`       HTTP 请求封装
- `pkg/db`         数据库连接
- `pkg/cache`      内存缓存
- `pkg/auth`       认证鉴权示例
- `pkg/env`        环境变量与环境判断
- `pkg/timer`      定时/延时工具
- `pkg/consts`     常量及枚举示例
